## Introduction to Database

- A database is an organized collection of data that can be stored, managed, and accessed easily.

## Types of Databases
### 1. Relational Database (SQL)
- Stores data in tables (rows & columns).

#### Examples
- MySQL
- PostgreSQL

```bash
| id | name  | age |
| -- | ----- | --- |
| 1  | Ankit | 22  |
| 2  | Rahul | 24  |
```

### 2. NoSQL Database
- Stores data in JSON-like format
- Document-based data

#### Examples
- MongoDB

```bash
{
  "name": "Ankit",
  "age": "25"
}
```
